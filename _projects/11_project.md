### Manimator

Text-to-Math-Animation generation using an **agentic workflow** and the Manim library. Describe what you want, get a mathematical animation.

**Links:** [GitHub](https://github.com/nabin2004/Manimator)

### Dualmation

A **Hybrid LLM-Diffusion Framework** for educational animation generation with RL feedback. Combines language model code generation with diffusion-based visual synthesis.

**Links:** [GitHub](https://github.com/nabin2004/dualmation) · [Paper (TeX)](https://github.com/nabin2004/Dualmation-paper)
